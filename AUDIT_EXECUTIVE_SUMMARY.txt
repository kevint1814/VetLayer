BUGS IDENTIFIED (5 CRITICAL)
================================

HIGH SEVERITY (Fix ASAP)
1. Partial Credit Calculation Bug (analysis.py:3479)
   - Issue: Depth 2 of required 3 with weight 0.5 gives only 20% credit
   - Impact: Undervalues almost-qualified candidates
   - Fix: Use min(partial_ratio, 1.0) * weight * 0.75 instead

2. Missing Implied Skill Validation (skill_pipeline.py)
   - Issue: LLM instructed to enforce "React implies HTML/CSS/JS at depth 3+"
   - Problem: No post-parse validation if LLM violates rules
   - Impact: Inconsistent skill depths
   - Fix: Add post-LLM validation to enforce minimum thresholds

3. Recency Factor Cliff Effect (analysis.py)
   - Issue: Sharp drop from 1.0 to 0.90 at 2-year boundary
   - Impact: Artificial discontinuity in scoring
   - Fix: Use smooth decay instead of brackets

4. Insufficient Cache Key (skill_pipeline.py:1014)
   - Issue: Cache misses if skill order changes or job_title slightly differs
   - Impact: Redundant LLM calls, slower performance
   - Fix: Normalize skill list and deduplicate in cache key

5. Transferability Transitive Bug (analysis.py)
   - Issue: System doesn't compute transitive transferability
   - Impact: Vue developers may not get Angular credit even though both relate to React
   - Fix: Implement transitive closure on skill adjacency graph

MEDIUM SEVERITY
6. Experience Range Not Validated
   - Job has experience_range but scoring doesn't check against it
7. Pydantic v2 Deprecation Warnings (3 files)
   - Convert class Config to ConfigDict
